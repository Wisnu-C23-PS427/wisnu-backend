# wisnu-backend

## How to run

1. Clone this repository

```bash
git clone https://github.com/mufidu/wisnu-backend.git
```

2. Install dependencies

```bash
cd wisnu-backend
pip install -r requirements.txt
```

3. Run the server

```bash
flask run
```

Server will run on port 5000

4. (Optional) Deploy to Google App Engine

```bash
gcloud app deploy
```
